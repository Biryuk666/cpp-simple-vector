# Упрощенный вектор

## Описание
**Упрощенный вектор** - это структура данных на базе C++, которая выполняет базовый функционал *std::vector*, но при этом итоговое приложение с ней занимает меньший объем памяти, что в условиях ограниченной памяти может быть критически важно. Поскольку **упрощенный вектор** работает с динамической памятью, то для предотвращения утечек, в нем присутствует собственный сборщик мусора.

## Возможности
* Метод *GetSize()* возвращает размер контейнера.
* Метод *GetCapacity()* возвращает вместимость контейнера.
* Метод *IsEmpty()* сообщает, пустой ли контейнер.
* Метод *Clear()* обнуляет размер контейнера, не изменяя его вместимость.
* Метод *Reserve(...)* изменяет вместимость контейнера, при условии, что новая вместимость больше, чем текущая.
* Метод *Resize(...)* изменяет размер контейнера. При увеличении размера новые элементы получают значение по умолчанию.
* Метод *PushBack(...)* работает как с *lvalue*, так и *rvalue* ссылками, и добавляет указанный элемент в конец контейнера.
* Метод *PopBack()* удаляет последний элемент контейнера.
* Метод *Insert(...)* работает как с *lvalue*, так и *rvalue* ссылками, и добавляет элемент в указанную позицию контейнера. Возвращаемым значением метода является итератор (указатель) на вставленный элемент.
* Метод *Erase(...)* удаляет элемент в указанной позиции контейнера и возвращает итератор (указатель) на следующий за удаленным элемент.
* Метод *Swap(...)* обменивает значения с указанным контейнером.
* Метод *At(...)* возвращает ссылку на элемент с указанным индексом. Если в метод передать невалидный индекс, выбрасывает исключение *std::out_of_range*.
* Оператор "[ ]" возвращает ссылку на элемент с указанным индексом, но не выбрасывает исключение если индекс невалидный.
* Методы *begin()* и *cbegin()* возвращают итератор (указатель) и константный итератор на позицию первого элемента контейнера.
* Методы *end()* и *cend()* возвращают итератор (указатель) и константный итератор на следующую за последним элементом позицию. Данные итераторы запрещено разыменовывать, поскольку это приведет к неопределенному поведению.

## Использование
Для использования **упрощенного вектора** необходимо скопировать файлы *array_ptr.h* и *simple_vector.h* в каталог с проектом и импортировать последний как стороннюю библиотеку.
